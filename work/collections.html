Svg.html
3 likes on Codepen

Svg.html is a collection of HTML 5 svg animations. It is intended to showcase the ease of various animations only using svg and CSS 3 keyframes. It is obvious that svg animations can be lightweight, modular and scalable on the web canvas. From simple buttons to charts to image drawing and fills, and even motion graphics demonstrations, HTML 5 svg animations can be used for all purposes.

Rams: Ten Principles of Good Deisgn
Inspired by Dieter Rams' Ten Principle of Good Design, I decided to make a web visualization using parallax effect with the ten principles. All Javascript codes are written by myself without the aid of libraries. The result is a stunning celebration of contemporary design projects in reference to these principles.
